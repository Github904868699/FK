1 sleep_time

一次动作完成后的sleep_time。

2 click_time

一次点击里“按下到松开”的持续时间。

3 threshold

模型检测的置信度阈值。

4 size

允许动作触发的范围（瞄准范围）。

5 scale

显示窗口的缩放比例

6 capture_x / 7  capture_y

屏幕捕获 ROI 的宽/高（单位：像素）。

8 target_fps

帧率

A toggle

展示窗口 deiconify() 和隐藏 withdraw()

B monitor

用于多显示器时选取显示器

C jitter

随机增加或减少click_time,让每次按下持续时间不完全一致
